<h1 align="center">EDU-TECH</h1>

## About

EDU-TECH aims to develop an advanced education platform that leverages AI technologies, such as GPT, to deliver unique and tailored learning experiences for users.

## Environment

**Node.js version 22 or later**. 
This repository is for hiring process, so it contains environment variables.

## How to run

1.  Install dependencies

    Server

    ```bash
    cd server
    npm install
    ```

    Client

    ```bash
    cd client
    npm install
    ```

2.  Run the application

    Server

    ```bash
    npm run start
    ```

    Client

    ```bash
    npm run dev
    ```

4.  You can optionally build the application

    Server

    ```bash
    npm run start
    ```

    Client

    ```bash
    npm run build
    ```
